Web development is a term used to refer to all the processes involved in the creation, building and maintenance of websites and web applications. A website is typically made up of two components; the frontend and the backend.

Frontend
The frontend involves the visual aspect of the website. It involves everything that the user can see and interact with.
Backend
The backend involves the "behind the scenes" operations that create the website's functionality.

Frontend Development

The primary objective of frontend development is to create the user interface of the website and ensure that it is responsive, accessible and visually appealing.

To achieve this, frontend developers use the following languages:

  1. HTML
  2. CSS
  3. JavaScript

Frontend developers make use of various frameworks to make the task of developing websites easier.

Backend Development

The primary objective of backend development is to develop the website's functionality. The backend is made up of three main components:

  1. Servers: They are used to process requests, host applications and manage resources.
  2. Databases: Used to strore, manage and retrieve data. Examples of database technologies include:
    • PostgreSQL
    • MySQL
    • SQLite
    • Microsoft SQL Server
    • MongoDB
  3. Application Programming Interface (API): A set of protocols that enables software applications to communicate

Various programming languages and frameworks are used for backend development
